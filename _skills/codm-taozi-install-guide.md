# CODM-Taozi Skill 安装与使用指南

## 安装方法

1. 打开 CodeBuddy IDE
2. 在对话框中输入：

```
安装 skill：https://github.com/wangqi422/catwang-llm-wiki/blob/main/_skills/codm-taozi.md
```

或者手动安装：
1. 复制 [codm-taozi.md](https://github.com/wangqi422/catwang-llm-wiki/blob/main/_skills/codm-taozi.md) 文件内容
2. 放入你项目的 `_skills/` 目录下

---

## 使用方法

直接输入 CODM 资产关键词，AI 会自动返回对应链接。

### 支持的关键词

| 输入 | 返回内容 |
|------|---------|
| **精修图** | 角色精修/三视图链接 |
| **黑铁** | 黑铁三视图链接 |
| **武器表** | 武器资产表（微信文档） |
| **已上线武器** | 已上线武器三视图 |
| **场景** | 场景资产路径 |
| **资产包** | 资产包大链接 |
| **截图** | 场景截图大链接 |
| **原画** | 原画组路径 |
| **特效** | 特效组路径 |
| **归档** | 宣发素材归档 |
| **Shotgun** | Shotgun 全局搜索 |

### 示例

```
你：精修图
AI：https://arthub.woa.com/codm/pan?node=120347513932340

你：武器表
AI：https://doc.weixin.qq.com/sheet/e3_m_WyRqcCpPKiug?...

你：场景截图
AI：https://arthub.qq.com/codm/pan?node=68723699860
```

还支持查询**团队联系人**（如"角色组长是谁"）和**武器颜色分级**（如"哪些能用"）。
