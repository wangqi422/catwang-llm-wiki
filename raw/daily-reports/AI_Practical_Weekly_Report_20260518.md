### 1. 即梦 Seedance 2.0 · 多模态 AI 视频生成器

3. **音频驱动**：上传背景音乐实现卡点生成

**核心语法**：使用 `@` 引用系统，如「参考 @Image1 的角色，用 @Video1 的镜头运动」

### 3. Midjourney V7 Omni Reference · 角色一致性革命
**一句话简介**：V7 版本推出的 **Omni Reference** 功能，允许用 `--oref [图片URL]` 锁定角色外貌，配合 `--cw` 参数控制一致性权重，彻底解决 AI 短剧「换脸」痛点。

**获取方式**：
- Discord /imagine 命令或网页端
- 需订阅 Midjourney 计划

**上手建议**：
1. **Step 1**：生成并保存角色参考图
2. **Step 2**：后续生成时添加 `--oref [URL] --cw 80-100`
   - `--cw 100`：严格保持脸部+服装
   - `--cw 0`：只保持脸部，服装可换
3. 配合 `--seedlock` 参数保持批次间一致性

*Source: @midjourney*

### 4. OpenAI Codex /goal · AI 编程的「长时域模式」
**痛点场景**：以前用 AI 写代码像指挥临时工——你说一句它干一点，干完就「摸鱼」等你下一条指令。

**AI 解决路径**：
Codex CLI 0.128.0 上线 `/goal` 命令，把 Agent 从「短跑选手」升级为「马拉松选手」：

```bash
# 设置持久目标
goal 迁移这个包到新 API 版本，并验证所有调用

# 长周期任务
/goal 把我 backlog.md 里的 18 个特性全干完

# 任务管理
/goal pause      # 暂停
/goal resume     # 恢复
/goal clear      # 清除
```

**预期效果**：
- 支持**跨会话、跨中断**的持久状态
- 空闲时自动继续，直到完成或预算用完
- 社区实测：18 个特性跑了 14 个，还自审代码、跑通 CI

*Source: @OpenAI, @sama*

### 5. Claude Code Native Git Worktrees · 多 Agent 并行互不打架
**痛点场景**：多个 Claude Code 窗口在同目录跑 → 改文件互相覆盖、git 状态混乱。

**AI 解决路径**：
Claude Code v2.1.108+ 原生支持 **Git Worktrees**，让每个 sub-agent 用独立临时 worktree：

```bash
# 配置 baseRef
worktree.baseRef: fresh    # 新功能：从最新 main 创建
worktree.baseRef: head     # Bugfix：从当前 commit 创建

# 委派时声明隔离
{
  "isolation": "worktree"
}
```

**预期效果**：
- PR squash-merge 后**自动清理**，不留孤儿分支
- 主目录永远干净，多 Agent 并行互不干扰
- 自动生成带上下文的 CLAUDE.md

*Source: @AnthropicAI*

### 6. 可灵 AI 3.0 · 数字演员绑定工作流
**痛点场景**：AI 短剧角色在不同镜头里「换脸」，观众出戏。

**AI 解决路径**：
可灵 3.0 的「数字演员」功能，把角色特征绑定为可复用资产：

4. **逐集生成**：每集调用绑定角色，保持同一张脸
5. **智能分镜**：自动调度各集镜头

**预期效果**：
- 3 集微短剧角色完全一致
- 生成效率比单镜头调试提升 3-5 倍

## 🧠 方法论与 Prompt (Methods & Prompts)

### 7. Kimi K2.6 Agent Swarm · 智能体集群编排
**核心理念**：把一个大任务拆成多个可并行子任务，由主 Agent 动态调度最多 **300 个子智能体**同时工作，执行时间缩短 80%。

**Prompt 模板**：
```
Role: 你是我的 AI 项目 orchestrator

任务: [一句话描述复杂任务]

自动执行:
1. 任务分解：识别可并行子任务
2. 子 Agent 分配：每个子任务一个专项 Agent
3. 并行执行：同时调度所有子 Agent
4. 结果汇总：整合所有产出为最终报告

输出格式:
- 各子任务执行状态
- 最终结果汇总
- 遇到的异常及处理方式
```

**实战案例**：
竞品调研任务 → 自动拆成 8-10 个竞品分支 → 各子 Agent 调研定价/功能/用户评价 → 4 分钟输出对比报告（单 Agent 需 15-20 分钟）

*Source: @Kimi_Moonshot, @yarazhangrui*

### 9. AI 视频 Prompt 五维公式 · 稳定出片 checklist
**核心理念**：从「抽卡盲盒」到「精准控制」，视频生成 prompt 需要结构化。

**Prompt 模板**：
```
[主体描述] + [动作细节] + [环境背景] + [光影材质] + [运镜方式]

示例:
赛博朋克风女战士(主体)，缓慢转头望向镜头(动作)，
雨夜霓虹街道倒影(环境)，侧逆光蓝紫色调 8K 分辨率(光影)，
缓慢推进特写(运镜)
```

**进阶技巧**：
1. **运镜语言**：推拉摇移跟 + 速度修饰词（缓慢/快速/匀速）
2. **光线控制**：黄金时段/阴天漫射/三点布光/霓虹背光
3. **负面提示**：避免「模糊、变形、多余肢体」

**实战案例**：
用 Seedance 2.0 生成 15 秒古风短片：
- 镜头 1：「古装女子缓步走过桃花林，衣袂轻扬，春日阳光」
- 镜头 2：「女子凭栏远眺，眼神悠远，微风拂面」
- 合成后添加鸟鸣、流水声
