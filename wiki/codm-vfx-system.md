# CODM 特效分类体系

> 使命召唤写实特效分 7 大类 110+ 子项，以六层爆炸系统和材质化命中反馈为核心特色，构成 FPS 游戏宣发视觉的技术标杆。

## Definition

CODM 特效分类体系是对使命召唤系列 CG/宣发视频中所有视觉特效的系统化分类，用于指导宣发素材制作中的特效参考和还原。

## Core Content

### 七大特效类别总览

| 大类 | 子类数 | 核心技术 | 重要度 |
|------|--------|---------|--------|
| 1.1 战斗与交互 | 20+ | 枪口焰/命中/弹道/贴花 | ★★★★★ |
| 1.2 破坏与物理 | 15+ | Voronoi碎裂/刚体物理/烟尘 | ★★★★★ |
| 1.3 烟火与爆破 | 20+ | 6层爆炸/火焰/烟雾/体积光 | ★★★★★ |
| 1.4 角色与群集 | 18+ | 面捕/毛发/布料/群集 | ★★★★☆ |
| 1.5 环境与天气 | 12+ | 天气系统/环境交互/自然现象 | ★★★★☆ |
| 1.6 光照与渲染 | 12+ | GI/光追/PBR/体积光 | ★★★★★ |
| 1.7 后处理与镜头 | 20+ | DOF/调色/状态效果/转场 | ★★★★☆ |

### 爆炸六层生命周期

```
Layer 1: 闪光核心 (1-3帧) — 极亮白/橙，HDR Bloom，照亮 10-30m
Layer 2: 火球 (0.3-1s) — 翻滚球形，核心橙→边缘暗红→黑烟边
Layer 3: 冲击波 (0.2-0.5s) — 球形扩散空气折射环
Layer 4: 碎片飞溅 (0.5-3s) — 初速 20-80m/s 径向弹射
Layer 5: 烟柱 (3-15s) — 浓黑向上翻滚，受风偏移
Layer 6: 余烬 (5-20s) — 橙色发光粒子，随风飘散
```

### 命中反馈链条

```
枪口焰 → 弹道轨迹 → 命中点特效 → 受击动画/音效 → 伤害标识
```

按材质分 7 类：水泥/金属/木材/泥土/雪地/玻璃/水面/人体

### 枪口焰差异化

| 枪型 | 焰型 | 烟量 | 交互光范围 |
|------|------|------|-----------|
| 突击步枪 | 紧凑锥形 | 中量快散 | 1-2m |
| 霰弹枪 | 大面积扇形 | 大量浓烟 | 3-4m |
| 狙击枪 | 长管/大口径爆闪 | 消音无/无消大量 | 2-3m |
| 轻/重机枪 | 持续大火焰 | 持续浓烟 | 2-3m |
| RPG | 尾焰喷射+前端闪光 | 巨量尾迹 | 大范围 |

### 拉片七维度分析法

镜头语言 / 光照氛围 / 粒子特效 / 材质表现 / 动画节奏 / 音效同步 / 后处理

## Key Points

- 爆炸六层是 COD 宣发视频的视觉核心——每一层独立可控
- 命中反馈的材质差异化是写实感的关键
- 拉片分析必须覆盖全部 7 个维度
- 破碎特效采用"离线预处理 + 运行时触发"两阶段模式
- 后处理状态效果（闪光弹/EMP/夜视仪）是 COD 独特视觉标识
