4 Claude Code 多 Agent 选型 · Subagents vs Agent Teams 多 Agent 协作
痛点：复杂任务塞给单个 Agent → 上下文爆炸；多开 Agent 又不知怎么分工。Anthropic 官方给出明确选型：能并行无依赖 → Subagents；需要讨论和交叉验证 → Agent Teams。实证：Code Review 覆盖率 16% → 54%。
Subagents（独立分身）
主派发 → 子 Agent 独立上下文完成 → 只回摘要。适合并行研究/独立任务
Agent Teams（对等协作）
多实例共享邮箱、互相讨论。适合架构师+实现者+审查者的博弈场景
# 最小落地模板 Step 1 主 Agent 读 README + CLAUDE.md，写任务拆解 Step 2 Task() 派发 3 个 Subagent：Explore / Plan / Implement Step 3 结果合并 → 交 Agent Team Review

7 Karpathy LLM Wiki · 让 AI 自我进化的个人知识库 252K 浏览
不要让 AI 每次从零翻你的 Obsidian / Notion——先让 LLM 把原始内容"编译"成结构化 Wiki（摘要 + 百科 + 双向链接），以后它只读 Wiki，Token 成本降 10×，答案一致性显著提升。原帖 252K 浏览 / 4.1 万收藏。
# LLM Wiki 知识库架构师 Prompt 目录约定 raw/ ：原始材料（只读） wiki/ ：结构化知识（平铺 + 标签） index.md ：索引（自动维护） log.md ：操作日志 加入新材料后执行 1) 生成摘要 → wiki/summaries/[title].md 2) 提炼知识页 → wiki/[slug].md 3) 更新 index.md + log.md 4) 添加 [[wikilinks]] 双向交叉引用 硬约束 - 每页 ≤ 150 行，根目录 ≤ 150 页 - Related 链接 ≤ 10（超限触发蒸馏）
Source: @karpathy · llm-wiki.md gist（2026.04.04）
8 Meta Prompting · 让 LLM 帮你写 Prompt 的脚手架 2026 行业共识
与其手写内容丰富的 Prompt，不如让 LLM 先生成结构化脚手架再填空。核心转变：从「内容驱动」到「结构驱动」。填一次脚手架 ≈ 过去改 5 轮 Prompt，且脚手架本身可沉淀为团队模板 10 倍复用。
# Prompt 架构师脚手架 你是 Prompt 架构师。任务：[一句话任务描述] 先不要给答案，按 6 要素生成 Prompt 脚手架： 1) ROLE AI 最合适的身份？ 2) TASK 任务精确边界？ 3) CONTEXT 必须提供的背景？ 4) EXAMPLES 几条 input→output 示范？ 5) OUTPUT 结构/字段/长度？ 6) CONSTRAINTS 禁止什么/必须什么？ 每一项都留成 [占位符]，由我来填。
实战案例
输入：「给 CODM 英雄 LTM 写 30s 短视频文案」→ 脚手架自动生成镜头脚本 JSON 格式 + 禁用词约束
沉淀价值
脚手架成为团队 Prompt 库，后续新任务直接复用
Source: @karpathy, @GaryMarcus · genaimlinstitute.com/blog/meta-prompting
9 Midjourney v7 电影感 Prompt 公式 · 3 段式 + 参数三件套 出图良率翻倍
MJ v7 已经不能再靠堆关键词"赌出图"——自然语言主导 + 结构化参数精修才是正解。公式 = 主体描述 + 镜头光线 + 风格 + --sref --cref --ar。弱 Prompt 10 张能用 1 张，结构化后 10 张有 6-7 张可进初审。
# Midjourney v7 结构化 Prompt 公式 [主体] 谁/什么/在做什么 + [环境] 具体地点/时间/天气 + [镜头] low angle / 85mm / anamorphic + [光线] golden hour / rim light / volumetric fog + [质感] shot on ARRI Alexa 65 / film grain, style of [导演/摄影师] --ar 21:9 --sref [风格锁] --cref [角色锁] --stylize 300 --v 7
--sref
锁风格，最大杀招
--cref
锁角色，人物一致性
--stylize
200 写实 / 400 偏艺术
Source: @midjourney, @alliekmiller · masonaibase.com/midjourney-v7-ultimate-guide-2026
