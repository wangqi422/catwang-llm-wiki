⚡ 高效工作流
3 套改变日常的工作方式
4 Midjourney V7 · Draft Mode + Omni-Reference 10x 迭代
痛点：AI 出图迭代慢、角色跨镜头变脸、风格不一致。MJ V7 的 Draft Mode 让概念迭代速度提升 10 倍（成本降低 50%），Omni-Reference 实现角色一致性锁定。
# MJ V7 Prompt 模板 [主体描述] + [环境/光线/镜头] + [风格关键词], --ar 16:9 --sref [风格图URL] --cref [角色图URL] --stylize 300 --v 7
过去
出 10 张图挑 1 张，等 10 分钟
现在
Draft 30 张选 3 张精修，3 分钟
∘ @midjourney
5 Claude Code Git Worktrees · 多 Agent 并行互不打架 自动清理
痛点：多个 Claude Code 窗口同目录跑 → 改文件互相覆盖、git 状态混乱。Native Worktrees 让每个 sub-agent 用独立临时 worktree，PR squash-merge 后自动清理。
# baseRef 配置 worktree.baseRef: fresh # 新功能：从最新 main 创建（推荐） worktree.baseRef: head # Bugfix：从当前 commit 创建 # 委派 sub-agent 时声明隔离 { "isolation": "worktree" }
新功能开发

Bugfix / 调试
baseRef: head
