AI 干货周报

AI 视频生成军备竞赛周 · 9 条立刻能用的干货

扫描范围 65 信源 · 精选 9 条 · 3 大板块 · 2026.05.12 – 05.18

这周是名副其实的 AI 视频生成工具军备竞赛周——字节 Seedance 2.0、Google Veo 3.1、可灵 AI 3.0 同时升级，多模态输入和角色一致性终于能让 AI 短剧落地了。

Seedance 2.0
四模态混合输入（文本+图片+视频+音频），生成 4-15 秒带原生音频视频，@引用系统精准控制
Veo 3.1
Google 首款视频+同步音频一体化生成，支持 1080p 和 16:9/9:16 双比例，首尾帧控制剧情
MJ V7
Omni Reference 锁定角色一致性，--oref [URL] --cw 80-100 彻底解决「换脸」痛点
Codex /goal
长时域模式上线，AI 编程从「临时工」变「马拉松选手」，跨会话持久执行 backlog

💡 一句话总结：AI 视频生成进入「多模态精准控制」时代，角色一致性不再是瓶颈；AI 编程进入「长时域自主执行」时代，Agent 能跑通宵了。

i. Tools & Apps 3 items
🛠️ 实用工具箱
3 个本周立刻能装的 AI 视频/图像工具
1 即梦 Seedance 2.0 · 多模态 AI 视频生成器 字节跳动
支持文本+图片+视频+音频四种模态混合输入，生成 4-15 秒带原生音频的高质量视频。角色跨镜头一致性碾压同级竞品，核心语法是 @引用系统。
Step 1
文生视频：用「主体+动作+环境+光影+运镜」五维结构写 prompt
Step 2
图生视频：上传角色参考图固定外貌，上传视频参考复刻镜头运动
Step 3
音频驱动：上传背景音乐实现卡点生成
∘ @BytedanceTalk · jimeng.jianying.com
2 Google Veo 3.1 · 带原生音频的视频生成 Google DeepMind
首次实现视频+同步音频一体化生成（环境音、音效、对白），支持 1080p 输出和 16:9/9:16 双比例。Gemini App 内直接使用。
提示词结构
[镜头运动] + [主体动作] + [光线氛围] + [音频描述]
核心功能
首尾帧生成控制剧情走向，Scene extension 拼接多片段
∘ @GoogleDeepMind · gemini.google.com
3 Midjourney V7 Omni Reference · 角色一致性革命 --oref
V7 推出的 Omni Reference 功能，允许用 --oref [图片URL] 锁定角色外貌，配合 --cw 参数控制一致性权重，彻底解决 AI 短剧「换脸」痛点。
# Step 1: 生成并保存角色参考图 # Step 2: 后续生成时添加参数 --oref https://example.com/character.jpg --cw 100 # 严格保持脸部+服装 --oref https://example.com/character.jpg --cw 0 # 只保持脸部，服装可换
∘ @midjourney · midjourney.com

4 OpenAI Codex /goal · AI 编程的「长时域模式」 马拉松选手
以前用 AI 写代码像指挥临时工——你说一句它干一点。Codex CLI 0.128.0 的 /goal 命令把 Agent 升级为「马拉松选手」：设置持久目标后，跨会话、跨中断自动执行，直到完成或预算用完。
# 设置持久目标 /goal 迁移这个包到新 API 版本，并验证所有调用 /goal 把我 backlog.md 里的 18 个特性全干完 # 任务管理 /goal pause # 暂停 /goal resume # 恢复 /goal clear # 清除
社区实测
18 个特性跑了 14 个，还自审代码、跑通 CI
核心升级
持久状态 + 自主循环，Agent 不再「7秒记忆」
∘ @OpenAI · Codex CLI 0.128.0
5 Claude Code Native Git Worktrees · 多 Agent 并行互不打架 自动清理
痛点：多个 Claude Code 窗口在同目录跑 → 改文件互相覆盖、git 状态混乱。Native Worktrees 让每个 sub-agent 用独立临时 worktree，PR squash-merge 后自动清理，不留孤儿分支。
# 配置 baseRef worktree.baseRef: fresh # 新功能：从最新 main 创建 worktree.baseRef: head # Bugfix：从当前 commit 创建 # 委派 sub-agent 时声明隔离 { "isolation": "worktree" }
∘ @AnthropicAI · Claude Code v2.1.108+
6 可灵 AI 3.0 · 数字演员绑定工作流 角色一致性
痛点：AI 短剧角色在不同镜头里「换脸」，观众出戏。可灵 3.0 的「数字演员」功能把角色特征绑定为可复用资产，3 集微短剧角色完全一致。
Step 1
建立角色库：生成主角正/侧/半身多角度图
Step 2
提取特征：用视频 3.0 Omni 提取角色动态特征
Step 3
绑定数字演员：保存为模板，逐集调用
∘ @Kimi_Moonshot · klingai.com

7 Kimi K2.6 Agent Swarm · 智能体集群编排 300 子 Agent
把大任务拆成多个可并行子任务，由主 Agent 动态调度最多 300 个子智能体同时工作，执行时间缩短 80%。适合竞品调研、技术选型、多语言内容生产。
# 每日自动简报 Prompt Role 你是我的 AI 项目 orchestrator 任务 [一句话描述复杂任务] 自动执行: 1) 任务分解：识别可并行子任务 2) 子 Agent 分配：每个子任务一个专项 Agent 3) 并行执行：同时调度所有子 Agent 4) 结果汇总：整合产出为最终报告
竞品调研
8-10 个竞品同时调研，4 分钟出对比报告
技术选型
Pinecone/Weaviate/Qdrant/Milvus 并行查文档
∘ @Kimi_Moonshot · kimi.com
8 DeepSeek V4 快速/专家双模式 · 按需切换推理强度 1M 上下文
V4 提供「快速模式」（轻量 V4 Lite）和「专家模式」（完整 V4 Pro）两种选择，1M tokens 上下文，日常对话用快速，复杂推理切专家。
快速模式
法律文本检索，秒级响应，给出具体操作表格
专家模式
物理仿真/数学推理，逻辑链完整，思考深度高
∘ @deepseek_ai · deepseek.com
9 AI 视频 Prompt 五维公式 · 稳定出片 checklist 从抽卡到控制
从「抽卡盲盒」到「精准控制」，视频生成 prompt 需要结构化五维公式。
# 五维公式 [主体描述] + [动作细节] + [环境背景] + [光影材质] + [运镜方式] # 示例 赛博朋克风女战士(主体)，缓慢转头望向镜头(动作)， 雨夜霓虹街道倒影(环境)，侧逆光蓝紫色调 8K 分辨率(光影)， 缓慢推进特写(运镜)
∘ @zarazhangrui · Seedance/Veo/Kling 实测
